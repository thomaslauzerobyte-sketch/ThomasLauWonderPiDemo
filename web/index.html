追踪参数

joints：start 时锁定肩/肘/腕基线，之后只动底座+腕 pitch；上下/左右都很快，且不会因 IK 失败卡住
点开始追踪后若发现"低头变抬头"就切换
joints 模式只在 start 时用一次（锁定肩/肘/腕基线）；ik 模式则每帧使用。z≤0.10 会让 pitch 向上时 IK 失败
点“开始追踪”后慢动到 home 的总时长，避免瞬间砸下来
